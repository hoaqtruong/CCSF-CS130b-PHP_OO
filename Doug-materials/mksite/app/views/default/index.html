You can use the generate.php script to create new controllers and models.

  • Copy mksite to your php directory.
  • Make a backup of ALL of your files before running generate.php.
  • Set the permission of generate.php to 755: chmod 755 generate.php.
  • Check the site in your browser: http://hills.ccsf.edu/~YOURNAME/index.php.
  • Create new controllers: generate.php controller sports nfl nba baseball soccer
  • Create new models: generate.php model scores
